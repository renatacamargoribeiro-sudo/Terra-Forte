## 🌱 Sobre o Projeto

Terra Forte é uma plataforma web moderna e responsiva que promove:

✅ **Reflorestamento** - Projetos de plantação e preservação de florestas  
✅ **Reciclagem** - Programas educacionais sobre reciclagem responsável  
✅ **Conservação Hídrica** - Iniciativas para preservação de água  
✅ **Energia Renovável** - Implementação de soluções de energia limpa  
✅ **Proteção de Ecossistemas** - Preservação de habitats naturais  
✅ **Educação Ambiental** - Programas educacionais sobre sustentabilidade  

## ✨ Características

### 🎨 Design Moderno
- Interface limpa e intuitiva
- Cores vibrantes que remetem à natureza
- Animações suaves e elegantes
- Totalmente responsivo (mobile, tablet, desktop)

### 🔄 Interações Dinâmicas
- **Navegação Suave**: Links com animações de scroll suave
- **Menu Responsivo**: Hamburger menu em dispositivos móveis
- **Galeria Interativa**: Modal para visualizar projetos em tamanho completo
- **Formulário de Contato**: Envio de mensagens com validação
- **Efeitos ao Scroll**: Elementos aparecem com animações ao rolar a página
- **Contadores Animados**: Estatísticas animadas sobre o impacto
